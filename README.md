📚 NumPy 기초 학습 (NumPy Fundamentals)
데이터 분석과 머신러닝의 기초가 되는 파이썬 라이브러리인 NumPy의 핵심 기능을 정리한 저장소입니다. 배열의 생성부터 다차원 연산, 슬라이싱까지 필수적인 문법을 포함하고 있습니다.

🗂 목차
배열의 형상(Shape) 확인

배열의 병합과 분리

배열 연산

배열 형태 변경 (Reshape & Transpose)

인덱싱 및 슬라이싱

주요 생성 함수

1. 배열의 형상(Shape) 확인
NumPy 배열의 차원과 크기를 이해하는 가장 기초적인 단계입니다.

1차원: (column,)

2차원: (row, column)

3차원: (layer, row, column)

2. 배열의 병합과 분리
병합 (np.concatenate): 여러 배열을 하나로 합칩니다.

분리 (np.split): 특정 인덱스나 축(axis)을 기준으로 배열을 나눕니다.

3. 배열 연산
배열 간의 산술 연산 및 통계 메서드를 활용합니다.

원소별 연산: +, * 등 (Element-wise operation)

통계 메서드:

sum(): 합계

mean(): 평균

min() / max(): 최소/최대값

cumsum(): 누적 합

4. 배열 형태 변경
데이터의 구조를 재구성할 때 사용합니다.

reshape: 배열의 요소 수는 유지하면서 차원과 크기 변경

T (Transpose): 전치 행렬 (행과 열을 바꿈)

5. 인덱싱 및 슬라이싱
다차원 배열에서 원하는 데이터에 접근하는 방법입니다.

a[1:4]: 1번부터 3번 인덱스까지 슬라이싱

a1[0, 1:3]: 특정 행에서 범위를 지정하여 추출

a2[:, 1, 1]: 3차원 배열에서 특정 축의 데이터만 추출

6. 주요 생성 함수
특정한 규칙을 가진 배열을 빠르게 생성합니다.
| 함수 | 설명 |
| :--- | :--- |
| np.zeros() | 모든 원소가 0인 배열 생성 |
| np.ones() | 모든 원소가 1인 배열 생성 |
| np.linspace() | 설정 범위를 균등한 간격으로 나누어 생성 |
| np.random.random() | 0~1 사이의 난수 배열 생성 |
| np.random.randn() | 표준정규분포를 따르는 난수 생성 |
